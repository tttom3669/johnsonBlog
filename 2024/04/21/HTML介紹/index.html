HTML介紹

HTML（HyperText Markup Language，超文本/字標記語言），實際上不是一種程式語言而是一種標記語言，可用來構造和網頁的整個內容。

HTML 是建立網頁的基礎，瀏覽器可以理解 HTML 文件，並將其渲染為最終的網站。

  • .html 是網頁共用副檔名，而 index.html 是伺服器在目錄中查找的默認文件 (首頁)。

HTML 標籤

HTML 使用「標記」（markup）來詮釋文字、圖像、或是其他能在瀏覽器裡面顯示的內容。

  • 使用標籤(Tag)來表示功能單位
  • 使用元素來描述不同類型的內容。賦予內容一些意義

使用巢狀結構來組合標籤

  • 層次感清楚，一層包一層

HTML 標籤架構

一個 HTML 標籤包含：

  1. 起始標籤(opening tag)：它包含了元素的名字，夾在一對角括號  <、> (brackets)之間。指明元素從何開始生效。
  2. 結束標籤(closing tag)：在標籤名稱前面還多加了一條 / 斜線 (forward slash) 。表示元素結束的地方。
  3. 內容(content):  元素的內容，不是必須包含。內容可以是純文字，或包含其他標籤 (子元素)。
    元素(element):  以上三者加起來就是元素 (HTML element)。

自我結尾標籤 (self-closing tag)

  • 本身同時是起始標籤和結束標籤。 在標籤結尾加了一條斜線，例如 <img />
  • 重要區別是自閉合標籤代表空元素(void element)，不能包含任何內容。

巢狀元素 Nesting elements

可以把元素(HTML element)放進另一個元素裡面，這叫做巢套(nesting)。

  • 元素的容器：父元素
  • 其中的元素被稱為子元素

HTML 標籤屬性、行為

每個 HTML 標籤都有各自可以設定的屬性與行為，屬性、行為會放在開頭標籤的角括號中

  1. 屬性(Attributes)用來描述元素的標籤的附加資訊(數據)，用來描述元素的信息，協助標籤運作。
  • 屬性名稱="屬性的內容"
  • 標籤可以包含一到多個屬性，多個屬性通常每組設定之間會留至少一個空格
  1. 行為(Methods) 用來設定元素的動態行為
